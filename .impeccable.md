# Pi Remote Web Design Context

## Users

Developers using the Pi coding agent to manage AI coding sessions. They use this dashboard to browse sessions, monitor active conversations, manage jobs, and interact with the agent in real-time. The context is typically during coding workflows, where focus and efficiency are paramount.

## Brand Personality

**Voice**: Technical, precise, uncluttered
**Tone**: Professional yet approachable for developers
**3-word personality**: dev, clean, minimal
**Emotional goals**: Evoke confidence in the agent's capabilities, provide calm focus during coding sessions, and deliver delight through smooth, efficient interactions.

## Aesthetic Direction

**Visual tone**: Clean, minimal, and technical - inspired by modern code editors like Zed
**References**: Zed editor's clean interface, VS Code's functional layout, but with Pi's distinctive personality
**Anti-references**: Avoid overly decorative elements, excessive animations, or anything that feels "designy" rather than functional
**Theme**: Dark mode primary (suitable for developer workflows), with light mode available

## Design Principles

1. **Function over form**: Every visual element must serve a clear purpose
2. **Developer-first**: Prioritize readability, efficiency, and keyboard-friendly interactions
3. **Subtle sophistication**: Use refined typography, precise spacing, and thoughtful color to create a premium feel without decoration
4. **Performance matters**: Fast, responsive interactions that never get in the way
5. **Consistency with purpose**: Maintain visual consistency while adapting to different contexts (mobile vs desktop, light vs dark)

## Technical Context

- **Framework**: SvelteKit + Svelte 5
- **Styling**: Tailwind CSS + DaisyUI with custom OKLCH color system
- **Typography**: JetBrains Mono (monospace) with system sans-serif fallback for chat
- **Current themes**: `pi-dark` (Tokyo Night-inspired) and `pi-light` (soft paper-white)
- **Components**: Session browser, live chat, job management, mobile-optimized layouts
